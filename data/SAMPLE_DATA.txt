================================================================================
DATA FORMAT:
================================================================================

- Register numbers are unique identifiers
- CGPA values are decimals (0.0 to 10.0)
- Salary packages are in LPA (Lakhs Per Annum)
- Technologies are comma-separated strings
- File format uses | (pipe) for field separation and ~ (tilde) for project fields
- Multiple projects are separated by ; (semicolon)


================================================================================
DATABASE TABLES:
================================================================================

students:
  - id, name, register_number, department, cgpa, year, batch

projects:
  - id, student_register_number, title, description, domain, technologies

companies:
  - id, name, eligible_cgpa, job_role, drive_date

placements:
  - id, register_number, name, department, batch, company_name, salary_package
