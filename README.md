Placement Management System — Core Java Console

A Java-based console application developed using Object-Oriented Programming (OOP) principles to manage students, companies, and placement records efficiently.

The system supports CRUD operations, automates candidate shortlisting based on CGPA and projects, and uses Java Collections along with file handling to ensure persistent data storage.

## 🚀 Features
- 👨‍🎓 Student Management (Add, View, Update, Delete)
- 🏢 Company Management
- 📋 Placement Record Tracking
- ⚙️ Automated Candidate Shortlisting (based on CGPA & skills/projects)
- 💾 Persistent Storage using File Handling
- 📊 Efficient Data Handling using Java Collections
- 🖥️ Menu-driven Console Interface

## 🛠️ Tech Stack
- Core Java  
- OOP Concepts  
- Java Collections Framework  
- File Handling
